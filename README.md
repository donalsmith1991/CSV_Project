# CSV_Project

This repository has been reorganized into a clean, beginner-friendly Python learning project structure.

## What changed

- `src/01_basics/` contains core Python fundamentals and early practice scripts.
- `src/02_file_io/` contains file handling, string operations, and CSV-related exercises.
- `src/03_projects/` contains real practice scripts and small projects from learning exercises.
- `src/04_misc/` contains miscellaneous utilities and additional practice code.
- `data/` contains sample data files used by exercises.
- `notes/` contains study notes and documentation for the project.
- `archive/` contains historical artifacts and screenshots that are not part of the core learning project.

## Additional cleanup

- `postman/` assets were moved out of this repository into a separate `Postman_Project` folder to keep this repo focused on Python learning.
- Old week folders were removed after the code was reorganized into meaningful folders with Pythonic snake_case names.

## Git workflow

This cleanup is being committed on a feature branch named `cleanup/reorganize-python-project` so the work can be reviewed and merged professionally.
